Olá mundo em C

Criado em 2024-09-16

Em C, para a gente receber e enviar mensagens para o compuador devemos utilizar a biblioteca padrão "stdio.h" - pois todas as funções já estão feitas para a gente.

O nome dessa biblioteca vem de "standard" (traduzindo ficaria "padronizado") e "input/output" - entrada e saída.

Mas enfim, para "printarmos" alguma coisa no bash, devemos primeiro... utilizar uma diretiva de preprocessamento chamada "include" - logo em seguida fazer uma função principal (no qual o compilador vai ser responsabilizado) - e por fim, utilizar a função "printf", que é da biblioteca que citei.

// File: main.c
// Date: 2024-09-14
// Author: oknauta
// License: GNU General Public License
#include <stdio.h>
int main()
{
    printf("Hello, world!");
    return 0;
}
